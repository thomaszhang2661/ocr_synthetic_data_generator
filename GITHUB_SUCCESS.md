## 🔗 使用方式

### 克隆项目

```bash
git clone https://github.com/thomaszhang2661/ocr_synthetic_data_generator.git
cd ocr_synthetic_data_generator
```

### 安装依赖

```bash
pip install -r requirements.txt
```

```bash

```

```bash
python quick_start.py mixed --samples 50
```
